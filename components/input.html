Inputs and validations

Add input class to the input element.

Protean also provides input validations. Add input-validationState to the input component for validation.

Additionaly, we can provide users with feedback message. Feedback message should have the validation-msg validationState classes.

Textarea, radio button and checkbox

Textarea allows the user to enter long text, such as comments. The user can drag and change the size of a textarea. Add text-area class to make it responsive.

Radio input allows the user to choose one item from list of items.

On the other hand, checkbox allows the user to choose multiple items from a list of items.
