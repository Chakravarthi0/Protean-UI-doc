Button

Buttons helps the user to perform an action with a single click.

Normal buttons

There are seven types of buttons. You can use btn class followed by btn btn-buttonType to create a button.

btn tags are designed to be used with <button> tag.

In addition to the seven type of buttons, you can create a button for links using btn-link class

Link

Outlined buttons

Use the btn class followed by btn-buttonType-ol class to create outlined buttons.

Icon buttons

Icons can be used as buttons. Wrap the icon inside a button with btn-icon class.

Sized buttons

Buttons have three sizes, large, medium and small. By default, buttons are of medium size. The size can be changed using btn-buttonSize class.

Floating action buttons

Floating action buttons are circular buttons, usually with a icon in the center.

Floating action buttons can be created using the following classes: btn , btn-buttonType-ol and btn-float
