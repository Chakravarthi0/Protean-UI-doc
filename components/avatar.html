Avatar

Avatar is used to display user profiles.

You can create a avatar using the avatar class, followed by avatar-size class. Avatar comes in four sizes: xs sm md and xl

Avatar with initials

If the user does not have a profile picture they can create an avatar just using their initials.

To create this type of avatar, you can use avatar class, avatar-size class, followed by avatar-initial class. By default the background color will be primary color. However, you can customise the alert using color and background color classes.
